  • /(_:_:)

    Joins a path and a string to produce a new path.

    Path.root/"a"       // => /a
Path.root/"a/b"     // => /a/b
Path.root/"a"/"b"   // => /a/b
Path.root/"a"/"/b"  // => /a/b

    See also

    Path.join(_:)

    Declaration

    Swift

    @inlinable
public func / <S>(lhs: Path, rhs: S) -> Path where S : StringProtocol

    Parameters

    lhs

    The base path to join with rhs.
    rhs

    The string to join with this lhs.

    Return Value

    A new joined path.
